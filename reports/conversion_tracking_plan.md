# Conversion Tracking Plan

Use direct tel links for visible call buttons and expose call-click support through /api/call-event. Use /api/lead for form submits. The lead endpoint is a safe placeholder and does not return submitted personal details in the response.

Add GA4, GTM, Clarity, Search Console, and Bing values through environment variables only after owner approval. Do not run live indexing, paid ads, or outreach automation from this repo.
