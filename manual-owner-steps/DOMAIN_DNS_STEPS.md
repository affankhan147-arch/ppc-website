# Domain And DNS Steps

Owner-only steps:

1. Choose the production domain.
2. Point DNS records to the approved hosting provider.
3. Confirm HTTPS is active.
4. Set `NEXT_PUBLIC_SITE_URL` to the final production URL.
5. Rebuild and redeploy after the final URL is configured.

Do not let Codex make payment, registrar, or DNS account changes without explicit owner approval.
