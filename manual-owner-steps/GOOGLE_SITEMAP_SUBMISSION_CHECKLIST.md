# Google Sitemap Submission Checklist

Use this only after Google Search Console ownership is verified.

## Checklist

1. Confirm the `plumbinghands.com` property is verified in Google Search Console.
2. Open Google Search Console:
   `https://search.google.com/search-console`
3. Select the `plumbinghands.com` property.
4. Open `Sitemaps`.
5. Submit:

```text
https://plumbinghands.com/sitemap.xml
```

If Google only asks for the path, submit:

```text
sitemap.xml
```

6. Check the sitemap status.
7. If the status says `Couldn't fetch`, wait and retry later.
8. Open `URL Inspection`.
9. Inspect:

```text
https://plumbinghands.com
```

10. Request indexing for the homepage.

## Important

Do not repeatedly spam indexing requests.

Do not request indexing for dozens of pages manually today unless a later Codex command asks for it.

Submitting a sitemap helps discovery but does not guarantee indexing.
