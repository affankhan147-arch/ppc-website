# Deployment Manual Steps

Codex prepared the app and deployment notes, but the owner must approve account-level deployment actions.

## Owner Steps
1. Choose hosting provider.
2. Add environment variables from `.env.example` with real owner-approved values.
3. Replace sample tracking phone number values.
4. Deploy from GitHub branch `master`.
5. Confirm sitemap and robots URLs after deployment.
6. Confirm forms and call tracking use owner-approved endpoints.

Do not add payment methods, DNS changes, or private credentials through Codex.
