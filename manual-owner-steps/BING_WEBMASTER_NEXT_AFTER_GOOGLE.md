# Bing Webmaster Next After Google

Do not execute this until Google Search Console is verified and the sitemap has been submitted there.

## Bing Setup

1. Open:
   `https://www.bing.com/webmasters`
2. Sign in with the owner-approved Microsoft account.
3. Add the site:

```text
https://plumbinghands.com
```

4. If Bing offers Google Search Console import, use it after Google is verified.
5. If import is not available, use manual verification.
6. Submit the sitemap:

```text
https://plumbinghands.com/sitemap.xml
```

## IndexNow

Do not run IndexNow yet.

Start IndexNow only after Bing Webmaster is verified and the sitemap is submitted.
