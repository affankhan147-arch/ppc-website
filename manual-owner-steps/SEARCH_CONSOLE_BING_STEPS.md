# Search Console And Bing Webmaster Steps

Owner-only steps:

1. Deploy the site.
2. Open Google Search Console.
3. Add the verified domain or URL-prefix property.
4. Add verification token through hosting environment variables or DNS.
5. Submit `/sitemap.xml`.
6. Open Bing Webmaster Tools.
7. Add the site and verify ownership.
8. Submit `/sitemap.xml`.

Do not commit verification secrets or account tokens to the repo.
