# Google Colab Integration Guide

## Purpose
Google Colab is the support lab for the PPC project. It prepares support files for Codex and the owner to review.

Colab is not the website builder, ad creator, payment tool, or credential store.

## Open The Notebook
Use:
`https://colab.research.google.com/github/affankhan147-arch/ppc-website/blob/master/colab/ppc_online_workflow.ipynb`

## Allowed Colab Outputs
- keyword CSVs
- page briefs
- content outlines
- summary files for Codex
- project support JSON

## Blocked Colab Activity
Do not use Colab to:
- store passwords
- store API keys
- store GitHub tokens
- create paid ads
- create Google Ads
- create Bing Ads
- create fake Google Business Profiles
- create fake reviews
- create fake addresses
- create fake licenses
- create fake insurance claims
- create fake local office claims
- create spam backlinks
- copy competitor images

## Safe Operating Flow
1. Open the notebook.
2. Run setup cells.
3. Generate support files under `colab-output/`.
4. Review the files.
5. Bring approved files back into GitHub through the normal local repo workflow.

## Owner Notes
If Colab asks for GitHub or Google authorization, the owner must decide and approve it manually. Do not paste tokens into notebook cells.
