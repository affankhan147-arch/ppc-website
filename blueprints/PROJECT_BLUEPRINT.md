# Project Blueprint

## Project
PPC lead generation platform focused on organic calls from SEO and AEO content.

## Primary Objective
Build a trustworthy, compliant, conversion-focused website and operating system that can generate inbound calls without paid ads as the first dependency.

## Core Pillars
- Organic SEO
- Answer engine optimization
- Clear call conversion paths
- Safe local/service-area claims
- Repeatable content operations
- GitHub-centered workflow control

## Current Phase
Workflow integration only. Website build starts after explicit Command 2 approval.

## Success Signals
- Workflow files committed to GitHub
- Manual owner steps clear
- Colab support notebook ready
- D-drive blueprint backup script ready
- Safety policy enforced before build work
