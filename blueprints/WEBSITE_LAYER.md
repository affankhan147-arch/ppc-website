# Website Layer

## Future Website Goal
Create a fast, accessible, conversion-focused website for organic call generation.

## Expected Future Components
- homepage
- service pages
- service-area pages where claims are accurate
- FAQ/AEO sections
- call-to-action blocks
- trust and compliance messaging
- analytics placeholders without credentials

## Build Rules
- Do not start build work during the workflow integration phase.
- Do not publish fake locations, fake credentials, or fake reviews.
- Do not copy competitor images.
- Use accessible markup and clear user flows when build begins.
